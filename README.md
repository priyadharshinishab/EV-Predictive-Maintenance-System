# 🚗 EV Predictive Maintenance System (AI-Powered)

An end-to-end AI-based system to monitor Electric Vehicle (EV) battery health, detect anomalies, predict failures, and send real-time alerts.

---

## 🔥 Features

- 🔍 Failure Prediction (Random Forest)
- ⚠️ Anomaly Detection (Isolation Forest)
- 🔋 Battery Health Prediction (Linear Regression)
- 📈 Temperature Trend Forecasting (LSTM)
- 🚨 Email Alert System (SMTP Integration)
- 📊 Interactive Dashboard (Streamlit)

---

## 🧠 Tech Stack

- Python
- Scikit-learn
- TensorFlow / Keras
- Pandas, NumPy
- Streamlit
- SMTP (Email Alerts)
